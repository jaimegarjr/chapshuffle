Chap Shuffle

Privacy Policy

Chap Shuffle is a Chrome extension that shuffles YouTube video chapters for a randomized viewing experience. This policy describes how the extension handles your data.

Data collection

Chap Shuffle does not collect, transmit, or share any personal data. The extension:

  • Does not make any network requests
  • Does not track your browsing activity
  • Does not collect analytics or usage data
  • Does not share any information with third parties

Local storage

The extension saves your preferences (shuffle enabled, minimum chapters, queue end behavior) using Chrome's built-in chrome.storage.sync API. This data is stored locally in your browser and synced across your Chrome profile by Google — it is never accessed by us.

Permissions

Chap Shuffle requests the following permissions:

  • activeTab — to read chapter data from the current YouTube page
  • storage — to save your extension preferences
  • youtube.com access — to inject the shuffle controls into the YouTube player

Contact

If you have any questions about this policy, you can open an issue on the GitHub repository.
